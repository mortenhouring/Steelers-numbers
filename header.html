Mobile Header Demo

This is a demonstration of the mobile header component. The header is fixed to the top and includes:

  • App title "Numbers game" on the left
  • Hamburger menu button on the right with outline and simple shading
  • Dropdown menu with navigation options

Tap the hamburger button to open the navigation menu. The menu is finger-friendly and positioned for easy mobile access.

All styling uses only black and white colors as specified, with no animations.

Navigation Options:

  • Current roster - Links to quiz1.html
  • New Faces - Links to newfaces1.html
  • Team Legends - Links to legends1.html
  • Scoreboard - Links to scoreboard.html

This header component can be easily integrated into existing pages by including the HTML structure and CSS styles.
